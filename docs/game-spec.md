# 学園祭投票ゲーム
学園生活シミュレーション × 維新の嵐風説得バトル。

## 技術方針
- HTMLのdiv/button/img + CSSスタイリングで全画面を構築する
- ポートレートは `<img>` タグで表示

## あらすじ
今まで学園祭は内部向けのイベントだったが、近隣の学園で外部公開の学園祭が始まり、
この学園でも外部公開するのかという話題が出てきた。
学園祭のあるべき姿が議論となり、学園祭の企画投票が始まる。

投票は30日後。学園生活を送りながら、生徒たちの支持を集めよう。

## 概要
プレイヤーは翠風学園の1生徒です。
派閥の支持者となって、自分の支持する派閥が投票で勝利するのを目指します。

## 3つの派閥（プレイヤーが1つを選ぶ）
| 派閥 | 中心人物 | 学園祭への立場 |
|------|----------|--------------|
| **保守派** | 鷹山 誠一（男・頑固。親は議員） | 外部公開反対。伝統的な内部イベントを守るべき |
| **革新派** | 結城 あかり（女・活発） | 外部公開賛成。学園を活性化 |
| **体育派** | 獅堂 剛（男・体育系。議論は苦手） | 学園祭を体育祭にし、他校と競い合うべき |

## 学園の規模と組織

### クラス構成（3学年 × 4クラス = 12クラス）
- 1年: 1-A, 1-B, 1-C, 1-D
- 2年: 2-A, 2-B, 2-C, 2-D
- 3年: 3-A, 3-B, 3-C, 3-D

### 部活
体育系：陸上部、サッカー部、野球部、テニス部
文化系：吹奏楽部、美術部

## 組織
それぞれのクラスと部活についてはそれぞれの組織のキーパーソンの支持でクラス・部活の支持が決まります。
投票当日を迎えて投票に勝つか、もしくは全ての組織の支持を統一したらゲームクリアとなります。
組織には性格があり、以下の4つのタイプがあります。
- 独裁型: 代表の支持が80%、副代表の支持が20%
- 衆議型: 代表の支持が40%、副代表の指示が40%、メンバーの指示が20%
- 委任型: 代表の支持が20%、副代表の指示が60%、メンバーの指示が20%
- 多数型: その組織の支持者の多い組織がその組織の支持となる。同数の場合は代表・副代表の支持できまる

例）数字は保守:革新:体育
組織が独裁型
代表が40:30:30
副代表1が20:50:30
副代表2が30:30:40
だとしたとき、
保守=40*80%+(20+30)/2人*20%=37
革新=30*80%+(50+30)/2人*20%=32
体育=30*80%+(30+40)/2人*20%=31
という計算から組織としては保守の指示となる

## 説得セッション仕様

### 前提条件
- 行動で「説得を挑む」を選択（体力消費なし。体力は態度選択時に消費）
- 1回以上会話した相手にのみ説得を挑める
- 同じ人物に対して何度でも説得を試みることができる

### 基本構造
- ターン制: 最大10ラウンド
- バーグラフ綱引き判定（-100〜+100。+70で成功、-70で失敗）
- 態度 → 話題 → 立場 の3段階コマンド構築（各段階でキャンセル可能）
- 好感度に応じてバーの初期位置に補正（好感度÷10）

### 説得の結果

#### 成功時（バー+70以上）
- 相手の思想パラメータがプレイヤーの支持する派閥の方向に **5〜20%** シフトする
- シフト量はバーの最終位置に比例する（+70で5%、+100で20%）
- 3軸の合計は常に100を維持する（増えた分だけ他の2軸から均等に引く）

#### 失敗時（バー-70以下）
- **プレイヤー自身の**思想パラメータが相手の思想方向に **5〜20%** シフトする
- シフト量はバーの最終位置に比例する（-70で5%、-100で20%）
- プレイヤーの支持派閥（最も高い軸）が変わった場合、**GAME OVER** となる

#### タイムアウト時（10ラウンド超過）
- バー位置に応じて双方の思想が **0〜10%** シフトする
- バー > 0: 相手の思想をプレイヤー方向にシフト（弱い勝利）
- バー < 0: プレイヤーの思想を相手方向にシフト（弱い敗北）
- バー ≈ 0: ほぼ変化なし（引き分け）

### Step 1: 態度を選ぶ
態度選択時に体力を消費する。体力不足時は行動がパスになる。

| 態度 | 効果倍率 | 体力消費 |
|------|---------|---------|
| フレンドリー | ×0.7 | ⚡3 |
| ふつう | ×1.0 | ⚡5 |
| 強気 | ×1.2 | ⚡8 |

#### 体力とパス
体力が低いと行動できずターンがパス（スキップ）になる確率が上がる。

| 体力 | パス確率 |
|------|---------|
| 20以上 | 0% |
| 10 | 約41% |
| 3〜5 | 約60〜70% |
| 3未満 | 100%（常にパス） |

### Step 2: 話題を選ぶ

| カテゴリ | 具体的な話題 | 特徴 |
|---------|------------|------|
| 思想（派閥） | 保守 / 革新 / 体育 | バーが大きく動く。機嫌-1 |
| 雑談（趣味） | 恋バナ / ゲーム / SNS 等 | 機嫌が変化する。バーへの影響は小さい |

**基本戦略**: 雑談で機嫌を温めてから思想話題で攻撃する。

### Step 3: 立場を選ぶ

| 立場 | 意味 | 有効な使い方 |
|------|------|------------|
| 肯定 | 賛同する | 自分の支持派閥を推す。運動ステータスが効く |
| 否定 | 反論する | 相手の支持派閥を攻撃する。知性ステータスが効く |

### 思想話題の効果計算

```
基礎効果 = 15 + (相手のその派閥への支持度 - 33) × 0.4
```

肯定も否定も、**相手の支持度が高い派閥ほど効果が大きい**。
- 自派閥を肯定: 相手の心に響くほど効果的
- 相手の最強派閥を否定: 核心を突くほど効果的

最終効果に以下の倍率を順に適用:

1. **態度倍率**: フレンドリー×0.7 / ふつう×1.0 / 強気×1.2
2. **機嫌倍率**: 激怒×0.3 / 不機嫌×0.6 / 平常×1.0 / 好意的×1.3 / 心酔×1.8
3. **ステータスボーナス**（思想話題のみ）:
   - 弁舌: `×(1.0 + 弁舌 × 0.008)` — 全般に有利（0→×1.0, 50→×1.4, 100→×1.8）
   - 運動: 肯定時 `×(1.0 + 運動 × 0.005)` — 肯定が得意なキャラ向け（0→×1.0, 50→×1.25, 100→×1.5）
   - 知性: 否定時 `×(1.0 + 知性 × 0.005)` — 否定が得意なキャラ向け（0→×1.0, 50→×1.25, 100→×1.5）
4. **好き属性ボーナス**（段階ボーナス方式）:
   - 一致0: ×1.00 / 一致1: ×1.10 / 一致2: ×1.20 / 一致3以上: ×1.25（上限）
   - 異性の外見系属性が1つ以上一致: 追加+0.05（最大×1.30）
5. **好感度倍率**: `×(1.0 + 好感度 / 200)`（好感度-100〜100 → ×0.5〜×1.5）

### 雑談の効果

雑談は主に**機嫌を変化**させる。バーへの影響は小さい。

| 条件 | バー効果 | 機嫌変化 |
|------|---------|---------|
| 相手の好きな趣味 + 肯定 | +3 | +2段階 |
| 相手の好きな趣味 + 否定 | -3 | -2段階 |
| 相手の嫌いな趣味 + 肯定 | -1 | -1段階 |
| 相手の嫌いな趣味 + 否定 | +1 | +1段階 |
| 普通/未知の趣味 + 肯定 | -1 | なし |
| 普通/未知の趣味 + 否定 | +1 | なし |

### 相手の機嫌（5段階）

| 機嫌 | プレイヤー効果倍率 | 反撃倍率 |
|------|----------------|---------|
| 激怒 | ×0.3 | ×1.5 |
| 不機嫌 | ×0.6 | ×1.2 |
| 平常 | ×1.0 | ×1.0 |
| 好意的 | ×1.3 | ×0.7 |
| 心酔 | ×1.8 | ×0.3 |

### 相手の反撃（毎ターン）

```
反撃 = (8 + 相手の弁舌 × 0.1) × 機嫌反撃倍率
```

### 日常パートとの連携
| 日常で得た情報 | 説得での効果 |
|--------------|------------|
| 趣味が判明 | 雑談で確実に機嫌を操作できる |
| 属性が判明 | 好き属性ボーナスの有無がわかる |
| 思想の数値が判明 | 肯定/否定の最適選択ができる |
| 好感度が高い | バー初期位置が有利になる＋効果倍率UP |

## 活動家システム
→ [activist-spec.md](activist-spec.md) を参照

各派閥から選出されたNPC（活動家）が毎日他の生徒を説得して回り、ゲームの情勢が動的に変化する。
活動家がプレイヤーをターゲットにした場合は説得バトルが強制的に開始される。
