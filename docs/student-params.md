# 生徒のパラメータ

## 能力値
- 弁舌：説得時に有利
- 運動：説得時の肯定に有利。ケンカで有利
- 知性：説得時の否定に有利
- 最大体力：体力の回復上限
- 体力：行動できる能力

## 性格
性格はキャラクターの個性を表す。説得バトルでの性格補正はなし（ステータスと属性相性で差別化）。
口癖や反応テキストに影響する。

| 性格 | 特徴 |
|------|------|
| 熱血 | 素直で情熱的。行動力がある |
| 慎重 | データ重視。慎重に物事を進める |
| 頑固 | 信念が強い。簡単には折れない |
| 柔軟 | 受け入れやすい。流されやすい一面も |
| 狡猾 | 裏で状況を読む。情報収集が得意 |

## 紹介
フレーバーテキストを100文字程度

## 属性

生徒の外見・雰囲気を表すタグ。3カテゴリから合計2〜4個を設定する。

### 髪型（必ず1つ選択）
ストレート、ポニーテール、ツインテール、三つ編み、ウェーブ、お団子、ボブカット
- 髪型は画像にも反映させる
- 男子はほぼストレート。女子はバリエーションを持たせる
- 髪型は属性(attributes)には含めず `hairStyle` フィールドで管理する

### 外見（複数可）
メガネ、金髪、幼い、大人、貧乳、巨乳
- 「幼い」と「大人」は排他（両方つけない）
- 「貧乳」と「巨乳」は排他。女子のみに設定する。未設定は普通体型を意味する
- キャラの年齢感・体格・イメージに合わせて設定する

### 性格・雰囲気（複数可）
陽キャ、陰キャ、真面目、不良、おしゃれ、天然、クール、元気、体育
- 「陽キャ」と「陰キャ」は排他
- 矛盾しない範囲で複数設定可（例: 真面目+メガネ、陽キャ+元気）

### 属性の付与ルール
- 合計2〜4個（髪型を除く）。多すぎると相性計算のバランスが崩れる
- キャラの description（紹介文）や性格に合った属性を選ぶ
- 外見属性は画像プロンプトにも反映するので、ポートレートと一致させる

## 好み/苦手属性

「異性（または他の生徒）のどんなタイプが好きか・苦手か」を表す。
好み属性・苦手属性ともに、髪型・外見・性格雰囲気の全カテゴリから選べる（`PreferenceAttr` 型）。

### 付与ルール
- 好み: 3〜5個（平均3〜4個）
- 苦手: 3〜4個（平均3個）
- 好みと苦手に同じ属性を入れない
- 自分の属性が苦手に入ることは許容する（例: 自分が「幼い」でも幼い異性が苦手）
- キャラの性格・紹介文から自然に導かれる好みを設定する
  - 例: 体育系キャラ → sporty を好む、真面目キャラ → delinquent が苦手
- ゲームバランス上、特定の属性に好み/苦手が偏りすぎないよう分散させる

## 好感度
- 範囲: -100〜+100（初期値0）
- 会話すると**必ず上がる**（マイナスにはならない）
- 基礎値5に相性倍率（×0.4〜×3.0）を掛ける → 結果は+2〜+15
- マイナスは攻撃（未実装）を受けた場合のみ想定

### 好感度の説得への影響
- バー初期位置: `好感度 ÷ 10`（好感度50 → +5から開始）
- 効果倍率: `×(1.0 + 好感度 / 200)`（好感度100 → ×1.5）

## 相性
相手の好み属性の中に自分の属性が入っているほど好感度が上がりやすく、
相手の苦手属性の中に自分の属性が入っているほど好感度が上がりにくい（ただし常にプラス）。

### 会話時の好感度上昇
```
上昇値 = 5 × 倍率（最低0.4、最大3.0）
好み属性が一致: 倍率 +0.4（1つにつき）
苦手属性が一致: 倍率 -0.2（1つにつき）
```

### 説得時の好き属性ボーナス
プレイヤーの好み属性と相手の属性の一致数 × 15% が効果に乗算される。
