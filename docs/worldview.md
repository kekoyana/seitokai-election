# 世界観・トーン設定

## 舞台

翠風学園（すいふうがくえん）──中規模の私立高校。3学年×4クラスの生徒が在籍する。
制服はネイビーブレザーに白シャツ、男子は赤ストライプネクタイ、女子は赤リボン。
校舎は3階建てで、教室・廊下・部室・グラウンドなどがある。

季節は9月。残暑が残る中、学園祭の企画投票が始まった。
外部公開するか、伝統的な内部イベントを守るか、あるいは体育祭に変えるか。
生徒たちはそれぞれの立場から、30日間の投票活動に臨む。

## 雰囲気

「放課後の教室」「昼休みの廊下」「部活帰りの夕暮れ」――
日常の学園生活の中に、政治的な駆け引きと青春の熱量が混ざり合う。
深刻すぎず、軽すぎず。生徒たちは真剣だが、あくまで高校生の日常の延長線上にある。

### トーンの指針
- ライトノベル寄りの青春群像劇。重厚な政治ドラマではない
- 生徒同士の距離感は近い。敬語と友達言葉が混在する
- ユーモアや軽口はあるが、茶化しすぎない
- 投票は本気だが、命がかかっているわけではない

## ゲーム内テキストのルール

### 基本方針
- **数値やシステム用語をそのまま見せない**。世界観の中の出来事として描写する
- 効果値（好感度の増減、体力の変動など）は**色付きテキスト**で簡潔に添える
- 主語はプレイヤー（一人称視点）。ただし「僕」「私」などの一人称代名詞は使わず、主語を省略する

### ログの書き方

#### 会話ログ
場所や時間帯を感じさせる短い描写 + 会話の成果。

良い例:
- `渡辺あおいと休み時間に話し込んだ。`
- `「読書って楽しいよね！」と嬉しそうに話してくれた。 ♥読書`
- `SNSの話題が出ると、あまり興味なさそうに目をそらした。 ✗SNS`

悪い例:
- `渡辺あおいと会話した。趣味「読書」が判明！（好き）` ← システム的すぎる
- `好感度 +5` ← 数値だけで世界観がない

#### 説得バトルログ
議論の場面なので、やや硬めの言い回しも許容。ただし「ダメージ」「HP」などのゲーム用語は避ける。

良い例:
- `疲れで頭が回らない…言葉が出てこない。`
- `少し息を整えた`

悪い例:
- `体力不足でうまく話せない...（パス / 体力+4）` ← メタ的すぎる

#### 効果表示の色
| 種類 | 色 | 用途 |
|------|------|------|
| ポジティブ効果 | `#7EC850`（緑） | 好感度UP、体力回復、趣味が好き |
| ネガティブ効果 | `#F07070`（赤） | 好感度DOWN、趣味が嫌い |
| 中立・軽微 | `#999` | 趣味が普通、変化なし |

効果テキストは `<span style="color:...">` で色を付け、描写文の末尾に添える。

### セリフのトーン（性格別）

| 性格 | セリフの特徴 | 例 |
|------|------------|-----|
| 熱血 | ストレート、感嘆符多め | 「よっしゃ！やるぞ！」 |
| 慎重 | 丁寧語まじり、分析的 | 「データで見ると…こうだと思います」 |
| 頑固 | 断定的、短文 | 「…行動で示せ」 |
| 柔軟 | 口語的、同調しやすい | 「えー、それめっちゃいいじゃん！」 |
| 狡猾 | 含みのある言い方 | 「ふふ、そうなんだ？」 |

### 避けるべき表現
- 「○○が判明！」 → 会話の中で自然にわかる描写にする
- 「パス」「ターン」「ラウンド」 → プレイヤーに見せるUIには使ってよいが、ログ文中では避ける
- 「ダメージ」「バフ」「デバフ」 → ゲーム用語は使わない
- 数値のみの羅列 → 必ず描写文を添える
