# 活動家（アクティビスト）仕様

## 概要
活動家とは、各派閥に所属し**自発的に他の生徒を説得して回るNPC**のこと。
プレイヤー以外にも説得活動をする生徒がおり、ゲームの情勢が動的に変化する。

## 活動家の人数

| 派閥 | 人数 |
|------|------|
| プレイヤーの支持派閥 | 1〜2名 |
| 他の派閥（各） | 3〜4名 |

合計で7〜10名が活動家として選出される。

## 活動家の選出条件

活動家になれるのは以下をすべて満たす生徒：
- プレイヤーではない
- 自派閥の思想値が**50以上**（3軸の中で最も高く、かつ50以上）

### 選出タイミング
- ゲーム開始時（初回）
- 毎日の開始時に再選出

### 解任条件
活動家の思想値が**40未満**に下がった場合、活動家から外れる。
条件を満たす別の生徒がいれば交代し、いなければ枠が空いたままになる。

### 選出の優先順位
思想値が高い順に選出する。同値の場合は弁舌が高い生徒を優先する。

## 活動家の行動

### 毎日1回、ターン終了時に行動
各活動家は1日に1回、他の生徒に対して説得を試みる。

### ターゲット選出
- 自派閥の思想値が最も低い生徒（＝切り崩しやすい相手）を優先
- ただし他の活動家はターゲットにしない
- **プレイヤーもターゲットになりうる**

### 説得の処理（NPC同士）
バトル画面は表示せず、内部で簡易判定を行う：

```
成功率 = 30% + (活動家の弁舌 - 対象の弁舌) × 0.3%
```

成功率は最低10%、最大60%にクランプする。

成功時：対象の思想を活動家の派閥方向に **3〜8%** シフトする。
失敗時：変化なし。

結果はログに記録する（例：「田中大輝が佐藤ゆいを説得した → 保守+5」）。

### プレイヤーがターゲットにされた場合
- **説得バトルが強制的に開始される**（通常のバトル画面で行われる）
- プレイヤーが「防御側」、活動家が「攻撃側」となる
- バトルのルールは通常の説得バトルと同じ
- バー初期位置は 0（好感度補正なし）
- プレイヤーが負けた場合（バー -70以下）：プレイヤーの思想が活動家の派閥方向にシフト
- プレイヤーが勝った場合（バー +70以上）：活動家の思想がプレイヤーの派閥方向にシフト
- 活動家がプレイヤーをターゲットにする確率は他の生徒と同等（特別扱いしない）

## ログ表示

活動家の行動結果は日常画面のログに表示する：
- NPC同士: `「{活動家名}が{対象名}を説得した（{派閥}+{値}）」` or `「{活動家名}が{対象名}の説得に失敗した」`
- プレイヤー対象: バトル終了後に通常の説得結果として表示

## UI表示

- 活動家であることはプレイヤーにはわからない（バッジ等の表示はしない）
- 日常画面のログで活動家の行動結果のみ確認できる
