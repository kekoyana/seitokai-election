# 相性の非対称性分析

生徒間の相性スコアは一方向的に計算されるため、A→BとB→Aで値が異なる。
ここでは差が大きい「片思い」ペアをまとめる。

## 相性スコアの計算式

```
score = (自分の属性がlikedに一致した数 × 10) − (自分の属性がdislikedに一致した数 × 8)
```

- プレイヤー属性は通常3個なので、理論上の範囲は **-24 〜 +30**
- コード: `debugScreen.ts` の `calcCompatibility()`、バトルでは `calcAffinityGain()` が同様のロジックを使用

## 最も差が大きいペア（差28）

| ペア | 片思い方向 | 好かれ側から見たスコア | 好き側から見たスコア |
|---|---|---|---|
| 佐藤ゆい ↔ 藤田芽衣 | 芽衣→ゆい | -8 | +20 |
| 渡辺あおい ↔ 小川はるか | あおい→はるか | +20 | -8 |
| 中村さくら ↔ 池田琴音 | 琴音→さくら | +20 | -8 |

### 佐藤ゆい ↔ 藤田芽衣
- 芽衣（sporty, fashionable, energetic, busty）→ ゆいはenergeticとfashionableが好き → +20
- ゆい（energetic_social, airhead, energetic）→ 芽衣はairheadが嫌い → -8
- 芽衣はゆいを気に入っているが、ゆいは芽衣をやや苦手に感じている

### 渡辺あおい ↔ 小川はるか
- あおい（introverted, cool, fashionable）→ はるかはfashionableとcoolが好き → +20
- はるか（airhead, young, flat）→ あおいはairheadが嫌い → -8

### 中村さくら ↔ 池田琴音
- 琴音（serious, glasses, adult, busty）→ さくらはadultとbustyが好き → +20
- さくら（airhead, fashionable, young, flat）→ 琴音はairheadが嫌い → -8

## 相互好意だが温度差があるペア（差20）

お互い好意的だが、片方だけが熱烈なケース。

| ペア | 熱烈側のスコア | 冷静側のスコア |
|---|---|---|
| 宮崎あかね → 山口涼 | +30 | +10 |
| 青木蒼 → 山口涼 | +30 | +10 |
| 井上桃花 → 木村ゆな | +30 | +10 |

山口涼は宮崎あかね・青木蒼の両方から+30で慕われているが、本人はどちらにも+10程度。
