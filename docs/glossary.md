# 用語集

## 登場人物

| 用語 | 説明 |
|------|------|
| 生徒 | 学園に所属する人物の総称。候補者もプレイヤーも含むすべての登場キャラクター |
| プレイヤー | ゲームの操作者。学園の1生徒として候補者を応援する |
| 候補者 | 生徒会長選挙に立候補する3名（鷹山/結城/獅堂）。学園内に出現するが説得はできない |
| 代表 | 組織内で最も影響力のある生徒 |
| 副代表 | 組織内で代表に次ぐ影響力を持つ生徒 |

### 表記ルール
- ゲーム内UIでは「人物」ではなく「**生徒**」を使う（候補者も生徒の一人）

## 思想・支持

| 用語 | 説明 |
|------|------|
| 思想軸 | 保守・革新・体育の3軸。合計は常に100 |
| 支持候補 | 思想軸の中で最も高い軸に対応する候補者 |
| 思想シフト | 説得の結果、思想パラメータが変動すること。成功で相手が、失敗でプレイヤーがシフトする |
| 保守 | 伝統を守る、秩序重視。候補: 鷹山誠一 |
| 革新 | 変革を求める、外部交流重視。候補: 結城あかり |
| 体育 | スポーツ・競争重視、実力主義。候補: 獅堂剛 |

## 組織

| 用語 | 説明 |
|------|------|
| 組織 | クラスまたは部活の単位。所属する生徒の支持で組織全体の支持が決まる |
| 組織タイプ | 支持の決まり方。独裁型/衆議型/委任型/多数型の4種 |
| 独裁型 | 代表80%、副代表20%で支持が決まる |
| 衆議型 | 代表40%、副代表40%、メンバー20%で支持が決まる |
| 委任型 | 代表20%、副代表60%、メンバー20%で支持が決まる |
| 多数型 | 組織内の多数派が支持となる。同数時は代表・副代表で決定 |

## 画面遷移

| 画面名 | コード上のscreen値 | 説明 |
|--------|-------------------|------|
| タイトル | `title` | ゲーム起動時の画面 |
| キャラ選択 | `candidate_select` | 支持候補・プレイヤーキャラを選ぶ画面 |
| 日常パート | `daily` | メインの探索・会話・説得を行う画面 |
| 説得バトル | `battle` | 生徒との説得セッション画面 |
| エンディング | `ending` | 全組織統一時の結果画面 |
| ゲームオーバー | `gameover` | 支持候補が変わってしまった時の画面 |

## 日常パート

| 用語 | 説明 |
|------|------|
| アクション | 1日に実行できる行動の単位。1日4アクション |
| 時間帯 | 朝/昼/午後/放課後の4つ。アクション消費で進行する |
| スタミナ | 行動に必要な体力。毎日100に全回復 |
| 移動 | 場所を変える行動。スタミナ消費なし |
| 会話 | 生徒と話す行動。スタミナ5消費。好感度UP＋情報が判明する |
| 説得 | 説得バトルを開始する行動。スタミナ20消費 |
| 休憩 | スタミナを30回復する行動 |
| 主義 | 生徒の生活スタイル。クラス派/部活派/帰宅派の3種で、よくいる場所が変わる |

## 活動家

| 用語 | 説明 |
|------|------|
| 活動家 | 各派閥に所属し、自発的に他の生徒を説得して回るNPC。プレイヤーの味方派閥から1〜2名、敵派閥から各3〜4名が選出される |
| 活動家の選出 | 自派閥の思想値50以上の生徒から、思想値・弁舌の高い順に選ばれる。毎日再選出 |
| 活動家の解任 | 思想値が40未満に下がると活動家から外れる。代わりがいなければ枠が空く |
| 強制説得 | 活動家がプレイヤーをターゲットにした場合、説得バトルが強制的に開始される |

## 説得バトル

| 用語 | 説明 |
|------|------|
| 説得セッション | プレイヤーと生徒が行うターン制の議論。最大10ラウンド |
| バー | 説得の進行度を表す綱引きゲージ（-100〜+100）。+70で成功、-70で失敗 |
| 態度 | プレイヤーが選ぶ話し方。フレンドリー/ふつう/強気の3択 |
| 話題 | 議論のテーマ。候補支持（保守/革新/体育）または雑談（趣味） |
| 立場 | 話題に対する姿勢。肯定/否定の2択 |
| 相手の態度 | 相手の機嫌を表す5段階。激怒/不機嫌/平常/好意的/心酔 |
| 反撃 | 相手ターンで自動的にバーが押し戻される |
| GAME OVER | 説得失敗の結果、プレイヤーの支持候補が変わってしまった状態 |

## 生徒のパラメータ

| 用語 | 説明 |
|------|------|
| 弁舌 | 説得時の基本的な有利さに影響するステータス |
| 運動 | 説得時の肯定効果に有利。ケンカにも影響 |
| 知性 | 説得時の否定効果に有利 |
| 性格 | 説得時の反応に影響する特性。熱血/慎重/頑固/柔軟/狡猾の5種 |
| 趣味 | 9種類の興味。各生徒ごとに好き/嫌い/普通が設定される |
| 好感度 | プレイヤーに対する好意の度合い（-100〜100）。説得効果に乗算される |

## 属性・相性

| 用語 | 説明 |
|------|------|
| 属性 | 生徒の外見・雰囲気を表すタグ。髪型/外見/性格雰囲気の3カテゴリ |
| 好み属性 | その生徒が好意を持ちやすい属性 |
| 苦手属性 | その生徒が不快に感じやすい属性 |
| 相性 | 自分の属性と相手の好み/苦手属性の一致度。好感度の変動量に影響する |

## マスクデータ（UIに表示しない内部パラメータ）

以下のデータはゲームロジックでは使用されるが、プレイヤーには表示しない。

| データ | 説明 | 使用箇所 |
|--------|------|----------|
| 性格 | 熱血/慎重/頑固/柔軟/狡猾の5種。説得時の反撃倍率・機嫌変動に影響 | 説得バトル |
| 組織タイプ | 独裁型/衆議型/委任型/多数型。組織の支持決定方式 | 組織票計算 |
| 属性 | 生徒の外見・雰囲気タグ（メガネ、金髪、元気など） | 好感度変動の相性計算 |
| 好み属性 | その生徒が好意を持ちやすい属性 | 会話時の好感度ボーナス |
| 苦手属性 | その生徒が不快に感じやすい属性 | 会話時の好感度ペナルティ |
| 会話回数 | プレイヤーとの会話回数 | 内部カウンタ |
| 好感度の数値 | 内部値は-100〜100の数値だが、UIでは文言で表示 | 下記参照 |

### 好感度の表示ルール

内部数値はプレイヤーに見せず、以下の3段階の文言で表示する。

| 表示 | 条件 | 色 |
|------|------|-----|
| 好意的 | 好感度 ≥ 20 | 緑（#27AE60） |
| 普通 | -20 < 好感度 < 20 | グレー（#888） |
| 不快 | 好感度 ≤ -20 | 赤（#C0392B） |

※ デバッグ画面（F2）では引き続き確認可能
