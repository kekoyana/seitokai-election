# 探索イベント仕様

散策の動機を増やすためのランダムイベント群。
移動や会話をトリガーに発生し、好感度やステータスに影響する。

## 1. 落とし物イベント

### 概要
部屋に入った際、一定確率で落とし物を発見する。
持ち主を探して届けると好感度が大幅に上昇する。

### 発生条件
- 部屋に入ったとき（`handleEnterRoom`）
- 発生確率: **15%**（1部屋あたり）
- 同時に所持できる落とし物は **1個まで**

### 落とし物の種類
| アイテム | 持ち主のヒント |
|----------|--------------|
| 手帳 | 名前入り → 持ち主が確定 |
| ハンカチ | イニシャル入り → 持ち主候補2〜3人 |
| お守り | 部活のマーク入り → 該当部活の誰か |
| ペンケース | クラスのシール → 該当クラスの誰か |

### フロー
1. 部屋に入ると「足元に何かが落ちている…」ポップアップ表示
2. アイテムと持ち主のヒントが表示される
3. `GameState.lostItem` に `{ itemName, hint, ownerId }` を保存
4. 持ち主がいる部屋に行き、話しかけると「届ける」選択肢が出現
5. 届けると:
   - 持ち主の好感度 **+15**
   - アクションログに記録
   - `lostItem` をクリア
6. 時間コスト: 届ける行為自体は **0分**（移動コストのみ）

### 制限
- 翌日に持ち越し可能（日をまたいでも消えない）
- 落とし物所持中も通常行動は制限されない

## 2. 会話情報収集

### 概要
生徒との会話時に、他の生徒に関する情報を教えてもらえることがある。
情報は「趣味の好み」「どの属性が好き/嫌いか」など、説得バトルで有利になるもの。

### 発生条件
- 会話（`handleTalk`）実行時
- 発生確率: **30%**
- 好感度が高いほど有用な情報が出やすい
  - 好感度 < 0: 発生しない
  - 好感度 0〜29: 趣味1つ解禁（対象の`revealedHobbies`に追加）
  - 好感度 30〜59: 趣味1つ + 好み属性1つ
  - 好感度 60+: 趣味2つ + 好み/苦手属性1つ

### 対象の選択
- 会話相手と **同じクラスまたは同じ部活** の生徒からランダムに選ぶ
- 既に全情報が判明済みの生徒は除外

### 表示
- 会話ウィンドウの最後に情報パネルを追加表示:
  - 「○○さんって、△△が好きらしいよ」（趣味解禁）
  - 「○○って□□な人が好みなんだって」（好み属性）
  - 「○○は××が苦手みたい…」（苦手属性）
- アクションログにも記録

### データ更新
- 対象生徒の `revealedHobbies` に趣味を追加
- 新フィールド `revealedLikes` / `revealedDislikes` をStudentに追加し、判明済みの好み/苦手を管理

## 3. おつかいイベント

### 概要
会話中にたまに「○○さんにこれを届けてほしい」と頼まれる。
届け先の生徒がいる場所まで行って届けると、依頼主と届け先の両方の好感度が上がる。

### 発生条件
- 会話（`handleTalk`）実行時
- 発生確率: **20%**
- 同時に受けられるおつかいは **1件まで**（既に受注中なら発生しない）
- 依頼主の好感度が **10以上** であること

### おつかいの種類
| 種類 | セリフ例 |
|------|---------|
| 手紙 | 「○○さんに手紙を渡してくれない？」 |
| ノート | 「○○にノート返しておいてほしいんだけど…」 |
| 文房具セット | 「○○の文房具セット、預かってるんだけど届けてくれる？」 |
| 伝言 | 「○○に伝えてほしいことがあるんだけど…」 |

### フロー
1. 会話終了時に「おつかい依頼」ポップアップ表示
2. 「引き受ける」/「断る」を選択
3. 引き受けた場合:
   - `GameState.errand` に `{ fromId, toId, itemName }` を保存
   - HUDまたは画面下部におつかいアイコン表示
4. 届け先の生徒がいる部屋に行き、話しかけると「届ける」選択肢が出現
5. 届けると:
   - 依頼主の好感度 **+8**
   - 届け先の好感度 **+8**
   - アクションログに記録
   - `errand` をクリア
6. 時間コスト: 届ける行為自体は **5分**

### 届け先の選択ルール
- 依頼主と **異なるフロア** にいる生徒を優先（散策を促す）
- 依頼主と同じクラスまたは同じ部活の生徒

### 制限
- 翌日に持ち越し可能（日をまたいでも消えない）
- 断ってもペナルティなし

### 発生時の会話演出
- おつかいイベント発生時、依頼主との会話が表示される
- 会話終了後に「引き受ける」/「断る」の確認ダイアログが表示される

---

## GameState への追加フィールド

```typescript
// 落とし物
lostItem: {
  itemName: string;
  hint: string;
  ownerId: string;
} | null;

// おつかい
errand: {
  fromId: string;
  toId: string;
  itemName: string;
} | null;
```

## Student への追加フィールド

```typescript
revealedLikes: PreferenceAttr[];    // 判明済みの好み属性
revealedDislikes: PreferenceAttr[]; // 判明済みの苦手属性
```
